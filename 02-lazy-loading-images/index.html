Example 02 - Lazy loading images

To see lazy loading in action:

  1. Open DevTools (Right Click → Inspect)
  2. Go to the Network tab
  3. Make sure "Disable cache" is checked (⚙️ settings in Network tab)
  4. Refresh the page and observe that images are not loaded immediately
  5. As you scroll down, watch the Network tab - new image requests will appear only when they come close to the viewport
